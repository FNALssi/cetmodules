<PROJECT-NAME>_ADD_NOARCH_DIRS
------------------------------

Specify project variables refering to non-architecture specific
locations. In the case of overlap with
:variable:`CETMODULES_DEFAULT_ARCH_DIRS`,
:variable:`!<PROJECT-NAME>_ADD_NOARCH_DIRS` governs.

.. seealso:: :variable:`<PROJECT-NAME>_ADD_ARCH_DIRS`, :variable:`CETMODULES_DEFAULT_ARCH_DIRS`.
