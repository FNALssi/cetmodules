<PROJECT-NAME>_ADD_ARCH_DIRS
----------------------------

locations. In the case of overlap with
:variable:`<PROJECT-NAME>_ADD_NOARCH_DIRS`,
:variable:`<PROJECT-NAME>_ADD_NOARCH_DIRS` governs.
